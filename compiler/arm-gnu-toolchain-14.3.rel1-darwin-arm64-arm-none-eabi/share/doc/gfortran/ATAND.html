8.24 ATAND — Arctangent function, degrees

Description:

ATAND(X) computes the arctangent of X in degrees (inverse of TAND).

Standard:

Fortran 2023

Class:

Elemental function

Syntax:
RESULT = ATAND(X)
Arguments:
XThe type shall be REAL; if Y is present, X shall be REAL.
YThe type and kind type parameter shall be the same as X.
Return value:

The return value is of the same type and kind as X. The result is in degrees and lies in the range -90 \leq \Re \atand(x) \leq 90.

Example:
program test_atand
  real(8) :: x = 2.866_8
  x = atand(x)
end program test_atand
